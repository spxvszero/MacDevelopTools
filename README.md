# MacDevelopTools
A Mac application for useful tools, build it just for learning :)

# Some Tips

* Right Click on Status Item can quit App

# Contains

* Apple Push
* Wallpaper
* Encode
* JsonToModel
* ResizeImage(not implenment)
* Move Files
* StatusIcon(not implenment)
* ImageInfo
* AutoPack(iOS)
* QRCode
